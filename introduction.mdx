## What fault can do

fault is organized into several feature areas, each accessible through its command group:

- **Moderation** — Full suite of moderation commands: warn, kick, ban (regular, hard, soft), timeout, mute variants, role management, nickname changes, and a complete case history system.
- **AutoMod** — Native Discord AutoMod integration. Enable anti-spam, block invite links, limit mass mentions, and manage a keyword blocklist — all managed through fault commands.
- **Tickets** — Multi-panel ticket system with topic dropdowns, priority levels, staff claiming, automatic transcripts, and log channels.
- **Leveling** — XP-based leveling with configurable multipliers, role rewards, custom level-up messages, and leaderboards.
- **Server Setup** — Welcome, goodbye, and boost messages; autoroles; custom prefix; sticky messages; paginated embeds; and fake permissions for fine-grained command access control.
- **Scripting** — A powerful embed script engine used by welcome messages, ticket messages, sticky messages, and more.

## Getting help

Use the `help` command inside Discord to browse all commands and categories. For support, join the fault support server using the link in the `invite` command response.
