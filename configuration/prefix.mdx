fault uses `,` as its default prefix. Change it to any string up to 10 characters.

## View the current prefix

```text
,prefix
```

Displays the prefix currently set for your server.

## Set a custom prefix

```text
,prefix set (prefix)
```

Requires the **Manage Server** permission.

```text
,prefix set !
,prefix set f!
,prefix set fault.
```

Prefix must be 10 characters or fewer.

## Reset to the default prefix

```text
,prefix reset
```

Returns the prefix to `,`.

You can always use a direct @mention instead of the prefix.

```text
@fault help
@fault prefix
```
