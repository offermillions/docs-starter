fault can post a message to one or more channels whenever a member joins your server, leaves it, or boosts it. Each message type is configured separately and supports the full fault scripting engine, so you can send plain text or richly formatted embeds.

<Note>
  All welcome, goodbye, and boost commands require the **Manage Channels** and **Manage Messages** permissions.
</Note>

<Tabs>
  <Tab title="Welcome">
    ## Welcome messages

    Welcome messages are sent whenever a new member joins your server. You can configure up to 3 channels to receive a welcome message.

    ### Add a welcome message

    ```text
    ,welcome add (channel) (script)
    ```

    The `(script)` is the message content. Use plain text or the fault scripting engine to build an embed.

    **Example — plain text:**

    ```text
    ,welcome add #welcome Hey {user.mention}, welcome to {guild}!
    ```

    **Example — embed:**

    ```text
    ,welcome add #welcome {embed}{description: Welcome {user.mention} to {guild}!}
    ```

    <Tip>
      The scripting engine gives you access to variables like `{user.mention}`, `{user.name}`, `{guild}`, and more. See the [scripting overview](/scripting/overview) for the full reference.
    </Tip>

    ### Manage welcome channels

    | Command | Description |
    |---|---|
    | `,welcome list` | List all channels that receive welcome messages. |
    | `,welcome view (channel)` | Show the current script for a specific channel. |
    | `,welcome remove (channel)` | Remove the welcome message from a channel. |
    | `,welcome clear` | Remove all welcome messages from every channel. |

    **Alias:** `welc` — for example, `,welc add #welcome (script)`.
  </Tab>
  <Tab title="Goodbye">
    ## Goodbye messages

    Goodbye messages are sent whenever a member leaves or is removed from your server. You can configure up to 3 channels.

    ### Add a goodbye message

    ```text
    ,goodbye add (channel) (script)
    ```

    **Example — plain text:**

    ```text
    ,goodbye add #goodbye {user.name} has left the server.
    ```

    **Example — embed:**

    ```text
    ,goodbye add #goodbye {embed}{description: Goodbye {user.mention}, we'll miss you!}
    ```

    <Tip>
      See the [scripting overview](/scripting/overview) for the full list of available variables and embed options.
    </Tip>

    ### Manage goodbye channels

    | Command | Description |
    |---|---|
    | `,goodbye list` | List all channels that receive goodbye messages. |
    | `,goodbye view (channel)` | Show the current script for a specific channel. |
    | `,goodbye remove (channel)` | Remove the goodbye message from a channel. |
    | `,goodbye clear` | Remove all goodbye messages from every channel. |

    **Aliases:** `leave`, `farewell` — for example, `,leave add #goodbye (script)`.
  </Tab>
  <Tab title="Boost">
    ## Boost messages

    Boost messages are sent whenever a member boosts your server. You can configure up to 3 channels.

    ### Add a boost message

    ```text
    ,boost add (channel) (script)
    ```

    **Example — plain text:**

    ```text
    ,boost add #boosts {user.mention} just boosted the server!
    ```

    **Example — embed:**

    ```text
    ,boost add #boosts {embed}{description: Thank you {user.mention} for boosting {guild}!}
    ```

    <Tip>
      See the [scripting overview](/scripting/overview) for the full list of available variables and embed options.
    </Tip>

    ### Manage boost channels

    | Command | Description |
    |---|---|
    | `,boost list` | List all channels that receive boost messages. |
    | `,boost view (channel)` | Show the current script for a specific channel. |
    | `,boost remove (channel)` | Remove the boost message from a channel. |
    | `,boost clear` | Remove all boost messages from every channel. |
  </Tab>
</Tabs>

## Scripting

All three message types use the same scripting engine. You can send a plain text message, or use script tags to build embed fields, set colors, add thumbnails, and more.

See the [scripting overview](/scripting/overview) for the full reference.
